== Description ==
Theme for Donky.cz site.

* Optimalized for simple e-commerce
* Usable only with odwp-donkycz-plugin installed
* The GPL v2.0 or later license. :) Use it to make something cool.

== Installation ==

1. Download the DonkyCz theme.
2. Unzip it in your themes folder.
3. In your admin panel, go to Appearance -> Themes and click the 'Activate' button to use the theme.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
